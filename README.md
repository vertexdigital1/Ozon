Создает сводную таблицу по кластерам из файла Excel.
    Работает с файлами, имеющими структуру отчета "остатки товаров на складах и в пути" Ozon.
    Аналитика - Отчеты - Управление остатками. 
    Для чего: некоторые планируют поставки по данному отчету, в сводном варианте - нагляднее. Почему скриптом: Потому, что в данном отчете надо много править ненужных ячеек и строк.
    Плюс, можно работать с файлами с разных кабинетов просто указывая путь к ним.
